AngularJS常用指令

内置指令

AngularJS中所有的内置指令前缀都为ng-,自定义指令不建议使用该前缀，以免冲突

ng-app

本指令用于声明Angular的边界 你可以用ng-app 指令告诉Angular 应该管理页面中的哪一块。

ng-model

本指令用于实现input和变量的双向绑定。

ng-bind-html

本指令是通一个安全的方式将内容绑定到 HTML 元素上。
当你想让 AngularJS 在你的应用中写入 HTML，你就需要去检测一些危险代码。通过在应用中引入 “angular-santize.js” 模块，使用 ngSanitize 函数来检测代码的安全性。

ng-init

本指令初始化应用程序数据，这个不常使用。通常情况下，不使用 ng-init。您将使用一个控制器或模块来代替它。

ng-controller

本指令用于为你的应用添加控制器。
在控制器中，你可以编写代码，制作函数和变量，并使用 scope 对象来访问。

ng-cloak

该指令用于在 AngularJS 应用在加载时防止 AngularJS 代码未加载完而出现的问题。
AngularJS 应用在加载时，文档可能会由于AngularJS 代码未加载完而出现显示 AngularJS 代码，进而会有闪烁的效果， ng-cloak 指令是为了防止该问题的发生。

ng-checked

本指令用于设置复选框(checkbox)或单选按钮(radio)的 checked 属性。
如果 ng-checked 属性返回 true，复选框(checkbox)或单选按钮(radio)将会被选中。

ng-show

本指令在表达式为 true 时显示指定的 HTML 元素，否则隐藏指定的 HTML 元素

ng-hide

本指令在表达式为 true 时显示指定的 HTML 元素，否则隐藏指定的 HTML 元素

ng-if

本指令用于在表达式为 false 时移除 HTML 元素。
如果if语句执行的结果为true，会添加移除元素，并显示。
ng-if 指令不同于 ng-hide，ng-hide 隐藏元素，而 ng-if 是从 DOM 中移除元素。

ng-switch

本指令根据表达式显示或隐藏对应的部分。
对应的子元素使用 ng-switch-when指令，如果匹配选中选择显示，其他为匹配的则移除。
你可以通过使用 ng-switch-default 指令设置默认选项，如果都没有匹配的情况，默认选项会显示

ng-change

本指令用于告诉AngularJS在HTML元素值改变时需要执行的操作。
ng-change 指令需要搭配 ng-model 指令使用。
ng-change 事件在值的每次改变时触发，它不需要等待一个完成的修改过程，或等待失去焦点的动作。
ng-change 事件只针对输入框值的真实修改，而不是通过 JavaScript 来修改。

ng-class

指令用于给 HTML 元素动态绑定一个或多个 CSS 类。
ng-class 指令的值可以是字符串，对象，或一个数组。
如果是字符串，多个类名使用空格分隔。
如果是对象，需要使用 key-value 对，key 为你想要添加的类名，value 是一个布尔值。只有在 value 为 true 时类才会被添加。
如果是数组，可以由字符串或对象组合组成，数组的元素可以是字符串或对象。

ng-repeat指令中使用trackby子语句解决重复数据遍历的错误

autofocus

获取焦点

ng-model

本指令绑定了 HTML 表单元素到 scope 变量中

ng-keyup

本指令用于告诉 AngularJS 在指定 HTML 元素上按键松开时需要的操作。

splice()

该方法用于插入、删除或替换数组的元素。

ng-keyup

指令用于告诉 AngularJS 在指定 HTML 元素上按键松开时需要的操作。
